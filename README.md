# LaboratoryOPI2
Лабораторна робота 2

Система DishGuide

FR-01: Система повинна дозволяти користувачу шукати ресторан за назвою або геолокацією.
FR-02: Система повинна надавати рекомендації щодо топ-страв ресторану на основі реальних відгуків.
FR-03: Система повинна відображати рейтинг ресторану на основі нагород та відгуків.
FR-04: Система повинна дозволяти користувачу фільтрувати ресторани за типом кухні або наявністю нагород.
FR-05: Система повинна дозволяти зареєстрованому користувачу зберігати обрані ресторани та страви.
FR-06: Система повинна надсилати користувачу сповіщення про нові ресторани у його місті.
FR-07: Система повинна дозволяти користувачу переглядати нові та популярні (трендові) заклади у його місті.
FR-08: Система повинна забезпечувати роботу через мобільний браузер із часом відповіді до 3 секунд для пошуку та 5–7 секунд для AI-рекомендацій.
FR-09: Система повинна бути інтегрована із зовнішніми сервісами для отримання даних про ресторани та аналізу відгуків.
FR-10: Система повинна надавати зареєстрованому користувачу персональні рекомендації на основі історії використання.
