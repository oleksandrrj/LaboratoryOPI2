# LaboratoryOPI2
Лабораторна робота 2

1. Система DishGuide

2. Функціональні вимоги

FR-01: Система повинна дозволяти користувачу шукати ресторан за назвою або геолокацією.
FR-02: Система повинна надавати рекомендації щодо топ-страв ресторану на основі реальних відгуків.
FR-03: Система повинна відображати рейтинг ресторану на основі нагород та відгуків.
FR-04: Система повинна дозволяти користувачу фільтрувати ресторани за типом кухні або наявністю нагород.
FR-05: Система повинна дозволяти зареєстрованому користувачу зберігати обрані ресторани та страви.
FR-06: Система повинна надавати персональні рекомендації користувачу.
FR-07: Система повинна відображати нові та популярні ресторани.
